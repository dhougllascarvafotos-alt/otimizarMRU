# MRU-Optimizer

Projeto de otimização de MRU (Most Recently Used)

## Estrutura do Projeto

```
MRU-Optimizer/
├── dados/              # Dados de entrada
├── src/                # Código fonte
├── resultados/         # Resultados das análises
├── mapas/              # Mapas e visualizações
├── relatorios/         # Relatórios gerados
├── requirements.txt    # Dependências do projeto
└── README.md          # Este arquivo
```

## Como Usar

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

2. Adicione seus dados na pasta `dados/`

3. Execute os scripts do projeto em `src/`

4. Verifique os resultados em `resultados/`, `mapas/` e `relatorios/`
